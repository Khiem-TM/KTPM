About Me

Welcome to the Library Management System! This platform helps you efficiently organize, track, and manage your library collection.

Library Management System Project by SCAR

Introduction:

  • A library management application designed to manage books, users, borrow/return transactions, real-time statistics, and reports.
  • The application is primarily developed in Java using the Spring Boot framework, Hibernate, MySQL, HTML, CSS, (JavaScript), and Bootstrap.
  • It is built based on the MVC (Model-View-Controller) design pattern.
  • The app features an integrated game and multi-threading for an enhanced user experience.

Feature Details:

Book Management:

  • Add new books
  • Delete books
  • Search for books in the library
  • Search for books using the Google Books API
  • View book lists
  • Generate book statistics

    • User Management:

    • Add new users
    • Delete users
    • Edit user information
    • Search for users
    • View user lists
    • Generate user statistics

For Users:

  • Borrow books
  • Return books
  • View and edit account information
  • Change password
  • Log in / Log out
  • Register a new account
  • Search for books
  • Create a favorites list
  • View borrowed books

Other Key Features:

  • Game: A book title guessing game with hints
  • Google Books API: Search for books using the Google Books API
  • Spring Security: Application security
  • OAuth2 Authentication: Login with Google / GitHub
  • Spring Websocket: Enables user-to-user chat
  • Spring Async: Integrated multi-threading for a smoother experience
  • Google Cloud Storage API: Store image data on Google Cloud Storage and save paths to the database
